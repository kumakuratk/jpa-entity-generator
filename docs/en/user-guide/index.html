Automatically generate relationships

In JPA, associations between entities can be automatically resolved by using annotations such as @OneToMany and @ManyToMany. The jpa-entity-generator automatically generates relationships between entities from foreign key information between DB tables. The specifications are explained here

.

JPA supports unidirectional and bidirectional relationships, but the relationships generated by jpa-entity-generator are only unidirectional.

Master transaction relationships

The two tables are master and transaction

  • One table (transaction) has a foreign key for another table (master)

  • The foreign key's referencing column (transaction.master_id) is not the table's primary key

Figure 1. Master transaction relationship table
Entities generated from master transactional tables
@Entity
@Table(name = "master")
public class MasterEntity {

  @Id
  private String id;

}

@Entity
@Table(name = "transaction")
public class TransactionEntity {

  @Id
  private String id;

  @ManyToOne(fetch = FetchType.LAZY)
  @JoinColumn(name = "id", insertable = false, updatable = false)
  private MasterEntity master;

}

parent-child relationship

  • One table (child) has a foreign key for another table (parent)

  • A column with a foreign key (child.id) is part of the table's primary key

Figure 2. parent-child relationship table
Entities generated from parent-child relationship tables
@Entity
@Table(name = "parent")
public class ParentEntity {

  @Id
  private String id;

  @OneToMany(fetch = FetchType.LAZY)
  @JoinColumn(name = "id", insertable = false, updatable = false)
  private Set<ChildEntity> children = new HashSet<>();
}

@Entity
@Table(name = "child")
public class ChildEntity {

  @Id
  private String id;

  @Column(name = "seq_no")
  private int seqNo;
}

many-to-many

@Entity
@Table(name = "left")
public class LeftEntity {

  @Id
  private String id;

  @ManyToMany(fetch = FetchType.LAZY)
  @JoinTable(
      name = "left_right_rel",
      joinColumns = {@JoinColumn(name = "left_id", nullable = false, updatable = false)},
      inverseJoinColumns = {
        @JoinColumn(name = "right_id", nullable = false, updatable = false)
  })
  private Set<RightEntity> rights = new HashSet<>();
}


@Entity
@Table(name = "right")
public class RightEntity {

  @Id
  private String id;

}
